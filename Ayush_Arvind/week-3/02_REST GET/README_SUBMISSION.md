# Spring Hands-On 2 - REST GET and MockMVC

Implements `/hello`, `/country`, `/countries`, `/countries/{code}` and MockMVC tests.

## Run
```bash
cd spring-learn
mvn test
mvn spring-boot:run
```
Test URLs: `http://localhost:8083/hello`, `http://localhost:8083/country`, `http://localhost:8083/countries`.
