# Student Course Portal - Angular Hands-On 1 to 10

This is the complete unified Angular v20 Student Course Portal solution for Digital Nurture 5.0. It implements all 10 hands-on tasks in one project, as required: components, data binding, lifecycle hooks, directives, pipes, template-driven forms, reactive forms, services, dependency injection, routing, lazy loading, guards, HTTP integration, interceptors, NgRx state management, effects, selectors, and Jasmine/Karma unit tests.

## Run

```bash
npm install
npm run api
npm start
```

Open `http://localhost:4200`.

## Test

```bash
npm test
npm run coverage
```

## JSON Server

`db.json` contains `courses`, `students`, and `enrollments`. Run it with:

```bash
npm run api
```
