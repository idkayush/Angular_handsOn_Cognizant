# Account, Loan and Eureka Hands-on

Start in this order:

1. `eureka-server`
2. `account-service`
3. `loan-service`

Run each project:

```powershell
mvn spring-boot:run
```

Test:

- Eureka: http://localhost:8761
- Account: http://localhost:8080/accounts/00987987973432
- Loan: http://localhost:8081/loans/H00987987972342
